将项目托管到码云中的步骤： 1.在项目根下建文件: .gitignore 将不托管的内容列在其中 2.git init 3.git status 查看需要托管的内容 4.git add . 准备添加到本地库 5.git commit -m "信息" 提交到本地库 6.注册一个码云帐号: 7.登录后进入设置,完成公钥等内容. 8.新建空项目 点+号新建,项目中不选且不添加任何内容 9.完成后,按照提示在命令方式下运行第一框和第三个的内容. 10.然后在我的码云中可以查看到托管的内容了. 11.如何以后对项目进行了修改,需要执行以下命令完成提交 1.git add . 2.git commit -m "提交信息" 3.git push 12.克隆文件到本地: git clone https://gitee.com/lcj_tmp/vue-element.git 13.webstorm如何用git在码云拉取项目，提交代码 1.在webstorm点击VCS ->check out from version control ->git 2.把代码上传到码云 3.选中项目，右键Git ->Repository ->push 写完之后可以看左下角 用命令行方式把修改过的代码上传到码云 1.git add . 2.git commit -m "提交信息" 3.git push
